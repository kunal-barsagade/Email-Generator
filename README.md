# Email Writer Assistant

Email Writer Assistant is an AI-powered application that generates professional email replies automatically.  
It integrates a Spring Boot backend, a React frontend, and a Chrome Extension to enhance email productivity.

## Project Overview

The goal of this project is to reduce the time spent writing repetitive or formal emails by providing. 
AI-generated responses directly inside the email workflow.

The system consists of:
- A **Spring Boot REST API** for email generation
- A **React frontend** for user interaction
- A **Chrome Extension** to integrate with Gmail

## Features

- AI-powered email reply generation.
- Tone-based email responses (professional, friendly, etc.)
- Chrome Extension integration with Gmail.
- Clean separation of frontend, backend, and extension layers

Tech Stack
Backend:
- Java
- Spring Boot
- REST APIs
- Maven

Frontend
- React (Vite)
- HTML, CSS, JavaScript
